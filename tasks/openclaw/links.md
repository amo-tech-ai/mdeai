## Notes

- **mdeai integration:** Paperclip `openclaw_gateway` adapter (see `tasks/prompts/05E-agent-infrastructure.md`) should align with **Gateway HTTP/WS** auth (`OPENCLAW_GATEWAY_TOKEN`, pairing) and channel config — not assumed Infobip-only; match your deployed gateway URL and tools.
- **Skills:** Prefer [ClawHub](https://github.com/openclaw/clawhub) / `openclaw skills` for discovery; the [skills](https://github.com/openclaw/skills) repo is an **archive** — verify skill safety before use.
- **Lobster:** Optional typed workflows; see [tools/lobster](https://docs.openclaw.ai/tools/lobster) and the [lobster](https://github.com/openclaw/lobster) repo.
