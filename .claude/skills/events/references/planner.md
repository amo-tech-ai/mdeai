# Phase 2 — Event planner (logistics)

> **Purpose.** Turn the brief into a 12-month → day-of timeline, vendor tracker, AV checklist, run sheet, and post-event evaluation plan. Logistics is the unglamorous work that makes or breaks the event.

## The 12-month timeline (compress for smaller events)

### 12+ months out (flagship only)

- Define event purpose, goals, success metrics
- Establish preliminary budget
- Select date (check holiday + competitor + venue availability)
- Form planning committee + assign roles
- Begin venue research and site visits
- Identify target audience + estimated attendance
- Determine event format (in-person / hybrid / virtual)

### 9–12 months out

- Book venue and sign contract
- Secure keynote speakers / entertainment
- Select and book caterer
- Begin sponsorship outreach
- Register event domain / create landing page
- Establish event branding (logo, theme, colors)
- Research and book AV vendor

### 6–9 months out

- Finalize budget allocations
- Open registration / ticket sales
- Book hotel room blocks (if overnight event)
- Hire photographer / videographer
- Plan menu with caterer (dietary accommodations)
- Design and order promotional materials
- Begin social media marketing campaign
- Recruit and train volunteers

### 3–6 months out

- Send formal invitations / email campaigns
- Confirm all vendor contracts and deposits
- Finalize event agenda and session schedule
- Arrange transportation / shuttle services
- Order signage, banners, name badges
- Set up registration management system
- Coordinate speaker logistics (travel, AV needs)
- Plan networking activities / icebreakers

### 1–3 months out

- Send reminder communications to registrants
- Finalize floor plan and seating arrangements
- Confirm final headcount with caterer
- Prepare printed materials (programs, handouts)
- Brief all staff and volunteers
- Create day-of run sheet (template below)
- Test all technology (AV, streaming, Wi-Fi)
- Finalize emergency and contingency plans

### Final week + day before

- Final vendor confirmations + venue walk-through
- Print name badges, registration packets, programs
- Confirm all speaker presentations received
- Send final logistics email to attendees
- Test AV equipment, microphones, projectors on-site
- Set up registration area, signage, wayfinding
- Brief volunteer team with assignments
- Charge all devices, prepare backup batteries

## Budget allocation by category

| Category | % of budget | Notes |
|---|---|---|
| Venue rental | 20–30% | Includes setup/teardown windows |
| Catering & beverage | 25–35% | Order 5–10% over confirmed headcount |
| AV & technology | 10–15% | Includes streaming if hybrid |
| Speaker fees / travel | 5–15% | Honorariums + flights + hotel |
| Marketing & printing | 5–10% | Promotion + day-of materials |
| Decor & signage | 3–8% | Branded environment |
| Entertainment | 3–10% | Music, hosts, performers |
| Photography / video | 2–5% | For post-event recap |
| Transportation | 2–5% | Shuttles, taxi vouchers |
| Insurance & permits | 1–3% | Required for most venues |
| Staff & volunteers | 2–5% | Stipends, t-shirts, meals |
| **Contingency** | **10–15%** | Don't skip — first surprise costs more than you'd think |

## Venue selection matrix (weighted scoring)

Score each venue 1–10 per criterion; multiply by weight; pick highest weighted total.

| Criterion | Weight | Why it matters |
|---|---|---|
| Capacity | 15% | Seated vs standing — different numbers |
| Location/access | 15% | Public transit + parking + Uber-friendly |
| Cost | 15% | Including AV + setup fees |
| AV capabilities | 10% | What's included vs add-on cost |
| Catering options | 10% | Exclusive caterer or BYO? |
| Parking | 10% | Capacity + cost to attendees |
| Ambiance / aesthetics | 5% | Photo backdrop quality |
| Accessibility (ADA) | 5% | Required, not optional |
| Wi-Fi / connectivity | 5% | Tested under load |
| Flexibility (layout) | 5% | Can rearrange between sessions? |
| Staff / service | 5% | Dedicated event manager? |

**Key questions to ask every venue:**

- Capacity (seated vs standing vs cocktail)?
- What is included in rental fee?
- Catering restrictions (exclusive vendor)?
- AV options + costs?
- Setup / teardown windows allowed?
- Cancellation policy?
- Insurance requirements?
- Outdoor backup options if weather risk?

**Contract essentials** — every vendor contract has:

- Scope of work (line-item)
- Pricing + payment schedule

## AV & technology checklist

### Core equipment

- ☐ Projector(s) and screen(s) — test resolution and brightness
- ☐ Laptop(s) for presentations + adapters (HDMI, USB-C)
- ☐ Wireless microphones (lavalier for speakers, handheld for Q&A)
- ☐ Sound system appropriate for room size
- ☐ Confidence monitor for speakers
- ☐ Timer / countdown display visible from stage
- ☐ Video recording setup (if recording sessions)
- ☐ Live streaming equipment (if hybrid event)
- ☐ Wi-Fi bandwidth tested under load
- ☐ Power strips and cable management
- ☐ Backup laptop and cables
- ☐ Presentation clicker / remote

### Hybrid / virtual additions

- ☐ Streaming platform selected and tested
- ☐ Camera(s) positioned for speaker AND audience
- ☐ Virtual attendee engagement tools (chat, polls, Q&A)
- ☐ Dedicated tech support person for virtual feed
- ☐ Breakout room capability for virtual attendees
- ☐ Recording and post-event access plan

### Dietary distribution (plan for)

```
Standard:        70–75%
Vegetarian:      10–15%
Vegan:           5–8%
Gluten-free:     5–8%
Dairy-free:      3–5%
Kosher / halal:  2–5%
Nut allergy:     2–3%
Other:           ask on registration
```

**Rules of thumb:**
- Order 5–10% more than confirmed headcount
- All buffet items clearly labeled with allergens
- Have a contact for custom dietary needs

## Post-event evaluation

### Attendee survey sections

```
1. OVERALL SATISFACTION (1–5):
   - Overall event experience
   - Value for money / time invested
   - Likelihood to attend again
   - Likelihood to recommend (NPS)

2. CONTENT & SPEAKERS (1–5 per session):
   - Relevance to your role
   - Quality of presentation
   - Actionable takeaways

3. LOGISTICS (1–5):
   - Venue quality and comfort
   - Food and beverage
   - Registration process
   - Signage and wayfinding
   - Technology / AV quality

4. OPEN-ENDED:
   - Most valuable part?
   - What should we improve?
   - Topics for next time?
   - Other feedback?
```

### Success metrics

| Metric | Formula | Target |
|---|---|---|
| Attendance rate | Attended / Registered | 80%+ |
| Net Promoter Score | %Promoters − %Detractors | 50+ |
| Overall satisfaction | Avg rating (1–5) | 4.2+ |
| Revenue vs budget | Actual / Budgeted | 100%+ |
| Cost per attendee | Total cost / attendees | track YoY |
| Social media reach | Impressions + mentions | benchmark |
| Leads generated | Qualified leads | goal-based |
| Sponsor satisfaction | Avg sponsor rating | 4.0+ |
| Session fill rate | Avg attendance / capacity | 70%+ |

### Post-event debrief agenda (90 min)

```
1. Metrics review (15 min) — attendance, revenue, NPS, survey highlights
2. What went well (15 min) — top 3 successes and why
3. What needs improvement (15 min) — top 3 issues + root causes
4. Vendor performance (10 min) — rate each vendor for future reference
5. Budget reconciliation (10 min) — final actuals vs budget, variances
6. Key decisions for next event (10 min) — changes, commitments, early bookings
7. Action items + owners (5 min) — every follow-up with deadline + owner
```

## Tool integrations

The frameworks above are tool-agnostic. If you operate the event in **Google Workspace** (the most common ops stack for organizers), wire them like this — adapted from the now-folded `persona-event-coordinator` skill.

### Prerequisites

Install these utility skills (or the `gws` CLI directly):
- `gws-calendar` — calendar entries with attendees
- `gws-gmail` — invitation + reminder emails
- `gws-drive` — upload event materials
- `gws-chat` — announce in Chat spaces
- `gws-sheets` — track RSVPs + logistics

### Workflow mapping

| Event lifecycle step | Google Workspace command |
|---|---|
| Create event in attendee calendars | `gws calendar +insert --location ... --attendee a@x.com --attendee b@y.com` |
| Long-range agenda view | `gws calendar +agenda --days 30` |
| Upload speaker decks + collateral | `gws drive +upload` |
| Send invitation emails | `gws gmail +send` (use templated body) |
| Announce updates to internal team | `gws workflow +file-announce` |
| Weekly status digest | `gws workflow +weekly-digest` |
| Pre-event meeting prep | `gws workflow +meeting-prep` |
| Track RSVPs + logistics | `gws sheets +append` (one row per booking / ticket) |

### Tips

- **Dedicated calendar per event series.** Don't pollute your personal calendar.
- **Bulk-invite via `--attendee` repeated** for ≤50 attendees. Above that, drive RSVPs through a dedicated form (e.g. mdeai.co's `/host/event/new`) and only put the **organizer team** on the master calendar entry.
- **Spreadsheet is source of truth for RSVP state**, but mirror to your CRM nightly.
